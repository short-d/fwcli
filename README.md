# fwCLI

Maintain microservices in your hands

## Roadmap

[Design doc](https://api.short-d.com/r/fwcli)

## Prerequisites

- Git v2.22.0
- Yarn v1.22.4

## Getting Started

1. Install the CLI
   
   ```
   go install github.com/short-d/fwcli
   ```

1. Generate repo for a new service

   ```
   fwcli new sample
   ```
